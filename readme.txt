== Description ==
Plugin Invelity Products Feeds is designed for Wordpress (WooCommerce) online stores and generates xml or csv files used for google/facebook or heureka feed.
Simply download the plugin and set up all necessary information in Settings/Reading.

== Installation ==

This section describes how to install the plugin and get it working.

1. Download the plugin and upload files directly via FTP (`/wp-content/plugins/invelity-products-feeds).

2. Activate the plugin through the 'Plugins' screen in WordPress.

3 In the main menu to the left you will see new menu item "Invelity Plugins" with a dropdown containing "Generate Feeds" settings page.

4 In Setting->Reading allow new taxonomies

5 If you want generate feed for all products let Products,Price filter, Products category filter empty.

6 Every merchant has own special input , don't forget fill it in.

7 After the settings are set up, you can proceed to generate feed file from your products.

8 List of all files is on "Manage Feed" where you can delete them if you need.

9.Files are automatically updated every day.If not, try reactivate plugin.

== Frequently Asked Questions ==

= Do I need anything else for this plugin to work? =

This plugin is pretty independent, but you should have Facebook, Heureka or Google account where you put files or links this plugin generate

You also need online store build using WooCommerce plugin.

= Is the plugin free? =

Yes. The plugin is free of charge.

We offer support, installation, setup and customization for a fee, contact us at https://www.invelity.com/ or directly at mike@invelity.com

= Can I choose specific Heureka/Google category or Brand to product?=
Yes.If you enable using Heureka/Google or Brand in settings each product will have new taxonomy for these categories.
You can add Heureka or Google Category for product category in settings for each Woocommerce product category also.
